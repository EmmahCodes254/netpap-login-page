HOW IT WORKS

Hotspot solution

NETPAP cloud allows you to sell/share your internet connection or start an ISP business. Payments are automatic via MPESA and All you need connect your router, load it with our conﬁgurations. Market to your neighbours and start earning money directly to your account.

How it works

  1. A user connect to the your Wi-Fi network
  2. They log In to the Captive portal to select bundle
  3. The User is the billed and sent an sms through the NETPAP cloud
  4. The user receives the voucher
  5. The user the access internet by using the sms voucher
